Меня зовут Дмитрий

С программированием я познакомился ещё в дошкольном возрасте, когда дедушка брал меня к себе на работу где были персональные компьютеры. Во время обучения в институте у меня появился свой собственный компьтер и я занялся изучением языка Java, что бы написать модификации для Minecraft. Хотя это не переросло в увлечение программированием, я понял, что благодаря ему можно сделать многое. Уже позже я решил попробавать зарабатывать таким образом на жизнь. Начал с разработки мобильных приложений. Но так как эта сфера достаточно узкая я перебрался в область веб-разработки. Где я занят по сегодня и планирую остаться и развиваться в будующем.

Навыки

Что я использую в работе

HTML5

Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта

CSS3

Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!

Java Script

Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое

Jquery

Библиотека Jquery позволит ускорить разработку. Без необходимости интегрировать в проект мы её не будем, но навык работы с ней присутствует

React

Эта библиотека позволяет создавать web-приложения. Мы можем создать максимально интерактивный продукт именно под ваши цели

Node.js

Эта платформа позволяет создавать бэкенд для вашего продукта - “мозги”, которые будут выполнять действия, которые пользователь не видит

PostgreSQL

Свободная объектно-реляционная система управления базами данных. PostgreSQL базируется на языке SQL и поддерживает многие из возможностей стандарта

Java

Строго типизированный объектно-ориентированный язык программирования общего назначения, разработанный компанией Sun Microsystems
